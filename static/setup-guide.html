myBotCoin

Setup Guide

From zero to a running Bitcoin bot in about 10 minutes. No Linux experience required.

What You'll Need
  • A Vultr account — cloud server hosting, ~$6/month  ·  vultr.com
  • A Kraken account — crypto exchange, free  ·  kraken.com
  • Your Kraken API Key & Secret — we'll create these in Step 2
  • About 10 minutes

Step-by-Step

1
Create Your Vultr Server
  1. Go to vultr.com and sign up or log in
  2. Click "Deploy New Server" (the blue + button)
  3. Choose "Cloud Compute — Shared CPU"
  4. Pick any location closest to you (e.g. Atlanta, Chicago)
  5. Select "Ubuntu 24.04 LTS x64" as the operating system
  6. Choose the $6/month plan (1 vCPU, 1 GB RAM — plenty)
  7. Set hostname to: myBotCoin-Bot
  8. Click "Deploy Now" — wait ~60 seconds for it to spin up
  9. Copy your server's IP address and root password from the Vultr dashboard
✓ You don't need to SSH into the server — myBotCoin installs itself automatically in Step 3.
2
Create Your Kraken API Keys
  1. Log in to your Kraken account at kraken.com
  2. Go to Security → API in the top-right menu
  3. Click "Add key"
  4. Name it: myBotCoin
  5. Enable only these two permissions:
    • "Query Funds" — to read your balances
    • "Create & Modify Orders" — to place trades
  6. Click "Generate Key"
  7. Copy and save both the API Key and API Secret — a notes app is fine
Important: Do NOT enable Withdraw permissions. The bot never needs them.
✓ Kraken only shows the API Secret once — copy it before closing the page.
3
Run the Auto-Installer
  1. Browse to dash.mybotcoin.io
  2. Click "New to myBotCoin? Set up your bot →"
  3. Enter your server's IP address and root password from Vultr
  4. Click "Install myBotCoin on My Server"
  5. Watch the log — installation takes about 60 seconds
  6. When you see "myBotCoin Installed!" — you're ready for Step 4
✓ You'll see lines scrolling by as Docker and myBotCoin install automatically. No Linux commands needed.
4
Connect Your Kraken Account & Configure
  1. After the installer finishes, a 3-step setup wizard will appear
  2. Step 1 — Paste your Kraken API Key and API Secret, then click Next
  3. Step 2 — Choose your settings:
    • Mode: Start with Auto — the bot reads the market and decides
    • DCA Amount: How much USD to invest on each scheduled buy
    • Aggression: Controls dip-buy sensitivity (start with Balanced or Ultra)
  4. Step 3 — Set a dashboard password
  5. Click "Finish Setup" — your bot is now live
✓ You can change any setting anytime from the dashboard — no reinstall needed.
5
Fund Kraken and Go Live
  1. In Kraken, go to Funding → Deposit
  2. Choose USD and select your deposit method (bank transfer, wire, etc.)
  3. Deposit at least enough to cover your first few DCA purchases
  4. That's it — the bot will execute your first buy at the scheduled time
✓ The bot only spends the amount you set. It will never exceed your DCA amount per scheduled buy.

Frequently Asked Questions

Do I need to keep my computer on?
No — the bot runs on your Vultr server 24/7. Your laptop can be off.
What if I want to stop the bot?
Go to Settings and switch Mode to Manual. It will stop placing trades immediately.
Is my Kraken account safe?
Yes — the API keys only allow trading, never withdrawals. The bot cannot move money out of Kraken.
How much does the Vultr server cost?
About $6/month. You can pause or destroy it anytime from the Vultr dashboard.
My DCA only bought half the amount I set — why?
The bot reserves a portion of your USD for dip buying (the Recycler pool). If your balance is close to that reserve, the DCA buy is reduced to protect dry powder. Add more USD to Kraken to resolve this.
What is the Recycler?
The bot's most powerful feature. It sells a portion of your BTC when it rises above your cost basis, then waits to buy back lower — ending up with more BTC than it started. These gains are "house money" that cost you nothing.
What is Auto mode?
The bot reads the 200-day moving average. If BTC is above it, the bot stacks BTC aggressively. If below, it protects value by accumulating USD. It switches automatically — you don't have to watch the market.
